# 📊 Data Science from Scratch – Learning Journey

This repository documents my journey of learning **Data Science from first principles** using the book *Data Science from Scratch by Joel Grus*.

Instead of relying heavily on libraries, this journey focuses on **understanding core concepts, mathematics, and algorithms by implementing them from scratch in Python**.

---

## 🚀 Objectives

* Build a strong foundation in **data science fundamentals**
* Understand the **math behind machine learning algorithms**
* Implement key algorithms **without relying on libraries**
* Apply concepts through **hands-on projects**
* Create a portfolio that demonstrates **real understanding, not just usage**

---

* Python for Data Science

## 🛠️ Approach

For each topic, I follow a structured approach:

1. Study concepts from the book
2. Write concise notes for clarity
3. Implement algorithms from scratch
4. Experiment with variations
5. Build mini-projects to apply learning
